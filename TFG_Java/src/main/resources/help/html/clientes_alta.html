Ayuda - Alta de Clientes

Formulario de Alta

En esta página puedes dar de alta un cliente en el sistema. Para ello, debes completar el formulario y presionar el botón "Alta".

Campos Obligatorios

Todos los campos son obligatorios.

Validación de Datos

El sistema valida que el nombre, apellido, teléfono y dirección no estén vacíos. El sistema valida que el teléfono sea numérico. El sistema valida que el formato del dni sea correcto.
