Ayuda - Alta de Citas

Formulario de Alta de citas

En esta página puedes dar de alta una cita en el sistema. Para ello, debes selecionar el cliente y el animal en los combos y presionar el botón "Nueva".

Campos Obligatorios

Cliente, Animal, Fecha, Hora. La descripcion es opcional.

Validación de Datos

El sistema valida tanto la fecha como la hora de la cita.
