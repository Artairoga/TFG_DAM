Ayuda - Alta de Animales

Formulario de Alta

En esta página puedes dar de alta un animal en el sistema. Para ello, debes ingresar los datos del animal en el formulario y presionar el botón "Guardar".

Campos Obligatorios

Nombre y dueño. El resto de los campos son opcionales.

Validación de Datos

El sistema valida que el nombre del animal no esté vacío y que el dueño sea un cliente existente en el sistema.
