Ayuda - Modificar Cliente

Formulario para Modificacion

En esta página puedes modificar un cliente del sistema. Para ello, debes selecionar el cliente en el combo y presionar el botón "Modificar".

Validación de Datos

El sistema valida que el nombre, apellido, teléfono y dirección no estén vacíos. El sistema valida que el teléfono sea numérico. El sistema valida que el formato del dni sea correcto.
